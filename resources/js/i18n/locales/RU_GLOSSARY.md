# Translation Glossary

This file contains the standardized translations for key terms used in this project. Its purpose is to ensure consistency across the application interface, logs, and documentation, regardless of who is working on the translations. Please refer to this list before adding or updating any localized strings.

## Terminology Mapping

| Source Term (EN) | Approved Translation (RU) | Context / Notes |
| :--- | :--- | :--- |
| **Inspect** | Изучить | Used for detailed analysis (e.g., analyzing logs or error details). |
| **Review** | Проверить | Used for verifying status, parameters, or configurations. |
| **alert channel** | канал уведомлений | Do not use "канал алертов" or "канал тревог". |
| **successful** | успех | Standardized status indicating successful operation completion. |
| **trusted** | доверенный | Applied to verified sources, certificates, or host nodes. |
| **backup** | резервная копия / резервное копирование | Use "резервная копия" for files/results. Use "резервное копирование" for the process itself. |

## General Rules
* Maintain a professional and technical tone across all localized elements.
